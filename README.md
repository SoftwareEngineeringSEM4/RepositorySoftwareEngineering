# CardGuard - Card Fraud Detection System

## Deskripsi Project
**CardGuard** adalah aplikasi inovatif yang dirancang untuk membantu pengguna melacak riwayat penggunaan kartu kredit secara real-time. Fokus utama dari aplikasi ini adalah memberikan lapisan keamanan tambahan dengan memanfaatkan teknologi untuk memeriksa apakah sebuah transaksi dikategorikan sebagai transaksi normal atau terindikasi sebagai tindakan penipuan (*fraud*). Dengan CardGuard, pengguna dapat merasa lebih aman dalam mengelola aktivitas finansial mereka.

## Penjelasan Fitur
Aplikasi ini memiliki beberapa fitur utama yang mendukung fungsionalitasnya:
* **Tracking History Transaksi:** Mencatat dan menampilkan riwayat penggunaan kartu kredit secara mendetail.
* **Deteksi Fraud Otomatis:** Sistem akan secara otomatis memeriksa setiap transaksi yang masuk dan memberikan status apakah transaksi tersebut aman atau mencurigakan.
* **User Authentication**: Fitur registrasi dan login untuk memastikan data kartu kredit hanya dapat diakses oleh pemilik akun.
* **Dashboard Monitoring**: Tampilan antarmuka (UI) yang memudahkan pengguna melihat ringkasan aktivitas kartu mereka.

## Cara Menjalankan/Menggunakan
Ikuti langkah-langkah berikut untuk menjalankan project ini di perangkat lokal Anda:

1.  **Clone Repository**: Salin repository ini ke komputer Anda.
2.  **Buka di Cursor**: Buka folder project menggunakan editor **Cursor** atau IDE pilihan Anda.
3.  **Persiapan Lingkungan**: Pastikan Anda telah menginstal dependensi yang diperlukan (seperti Python atau Node.js sesuai dengan teknologi yang digunakan).
4.  **Running Aplikasi**:
    * Jika menggunakan web server lokal, jalankan perintah: `[masukkan perintah running di sini, misal: npm start atau python app.py]`
    * Atau buka file `index.html` langsung melalui fitur *Live Server*.
5.  **Akses Aplikasi**: Buka browser dan arahkan ke alamat `localhost` yang sesuai untuk mulai menggunakan aplikasi.
